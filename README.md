# C

Well, `c` is THE BEST cli tool to clear your terminal!
When you launch `c`, you'll see your dude at the top of the terminal, so you won't feel so lonely

## Installation

To use this piece of art, just execute the following commands:
```bash
git clone https://github.com/EnotInc/c.git
go install
```

And now you can use `C`!

## Configuration

Yeah! You read it correctly! You can config c to suit your needs!
Here is a list of flags: 

**Help**
```bash
c -h
c --help
```

**Set your own dude**
```bash
c -d <string>
c --dude <string>
```

**Set to default**
```bash
c --reset
```
